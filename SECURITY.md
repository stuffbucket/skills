# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public issue for security vulnerabilities.**

Instead, please contact us at <contact@stuffbucket.com> with details of the vulnerability.

## Scope

This policy applies to skills and plugin configurations in this repository. For vulnerabilities in MCP servers or other external dependencies, please report to the respective maintainers.
